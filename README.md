
# Assignment

Movie & TV Show browsing app using the OMDB API


## 3rd party libraries

This project is using 3rd part libraries

* **Retrofit** - A type-safe HTTP client for Android, making network requests to REST APIs and automatically converting responses into Kotlin objects.
* **Glide** - Image loading library
* **Room** - Library for managing local SQLite databases in Android
* **Coil** - Image loading library for compose
* **Hilt** - A dependency injection framework for Android, built on top of Dagger
* **Gson** - A library that simplifies the conversion between Kotlin objects and JSON,
* **Mockito & Junit** - A mocking framework used in unit testing & framework for writing and running unit tests in Java and Kotlin
